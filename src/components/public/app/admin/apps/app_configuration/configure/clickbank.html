Setup Instructions

Follow these steps to install the {{integration.name}} app.

Step 1

Go to Clickbank and set up your product. You can use your own thank you page or paste SM thank you page URL here

Step 2

Click on Settings -> My Site

Step 3

Click on Edit in the Advanced Tools section

Step 4

Copy the ClickBank IPN URL and paste it here. Set your secret key to "IMBINC". Press Save Changes after you are done

Step 5

Go back My Products and copy the product ID

Step 6

Go to your SmartMember admin area, click on Build -> Access levels. Add new access level or edit existing one

Step 8

Go to the ClickBank Settings and paste your ClickBank Product ID into the textbox. Save your changes after you are done
